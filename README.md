# Ykt.ru | Система создания и учёта заявок-пропусков

Веб-система для учёта заявок-пропусков на вынос материальных ценностей из офиса. Позволяет создавать заявки-пропуска, отслеживать их статус, вести учёт.

## Возможности

- **Создание заявок-пропусков** с возможностью указания нескольких позиций оборудования
- **Отслеживание статусов**: «В работе», «Вынесено», «Возвращено»
- **Управление пользователями** (администратор может создавать/редактировать/удалять пользователей)
- **Статистика** по заявкам с детальным просмотром в модальном окне
- **Печать заявки** в формате документа
- **Экспорт в Excel** (XLS) всех заявок или выбранной категории
- **Авторизация** с базовой HTTP-аутентификацией

## Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/your-username/ykt-propusk.git
cd ykt-propusk
make up
```
### При первом запуске автоматически:

Сгенерируется файл .env со случайными паролями

Поднимутся контейнеры web (Apache + PHP) и MySQL

Инициализируется база данных

### После запуска сайт будет доступен по адресу:
#### http://ваш_домен:8080

#### Логин по умолчанию

#### Администратор: admin / password

#### Пользователь: user / password
